A responsive travel blog website created as a frontend learning project.
The goal of this project is to practice modern HTML, CSS, and JavaScript, as well as clean project structure and reusable components.

Technologies Used
HTML5 — semantic markup
CSS3
Flexbox
CSS Grid
BEM methodology
JavaScript (ES6+)
SVG icons (inline & reusable)
Git & GitHub

Features
Responsive layout (mobile-first approach)
Semantic HTML structure
Reusable components (social media block, cards, sections)
SVG icons with hover effects
Image sliders for categories and social media
Video section
Clean and scalable project structure
Prepared for dynamic content rendering via JavaScript

Social Media Block
The social media icons are implemented as a separate reusable component and used both in the header, left-post and footer.
